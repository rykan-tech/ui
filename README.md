# @rykan/ui

[![NPM](https://img.shields.io/npm/v/@rykan/ui.svg)](https://www.npmjs.com/package/@rykan/ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @rykan/ui
```

## Usage

```tsx
import * as React from 'react'

import MyComponent from '@rykan/ui'

class Example extends React.Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
